#TSV longest 2 char substring finder (coding test)

##Goal
Реализовать функцию принимающую на вход строку и возвращающую самую длинную подстроку состоящую не более чем из двух различных букв.

## HOWTO

### Build
``mvn package``

Requirements:
* _JDK 8 or higher_

### Run
``java -jar l2charsf.jar``
